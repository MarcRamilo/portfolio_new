Hamburgueseria Freser - Projecte HTML i CSS

Aquest projecte representa el meu treball en la creació d'una pàgina web per a una hamburgueseria fictícia anomenada "Freser". Vaig utilitzar HTML per estructurar la pàgina i CSS per donar-li estil, assegurant-me que la informació estigués organitzada de manera clara i efectiva.

Hamburgueseria Freser és un establiment fictici que ha implementat el servei de correu electrònic a través de formspree.io. Això permet als clients i visitants comunicar-se amb la hamburgueseria de manera fàcil i còmoda mitjançant formularis en línia.

Durant aquest projecte individual, l'objectiu era crear una pàgina web corporativa per a la hamburgueseria fictícia. Vaig treballar en diferents parts del desenvolupament utilitzant branques en un sistema de control de versions, i vaig gestionar el repositori remot a GitHub.

La metodologia de treball inclouia fer un pull al principi de cada sessió per mantenir els arxius actualitzats, i al final de cada sessió, fer un push al repositori remot per tenir-ho llest per a la següent sessió. Finalment, es realitzava un merge per concloure el projecte.

Enllaç al projecte: Ticketing

La temàtica central d'aquesta aplicació és la gestió d'esdeveniments i les vendes d'entrades associades a aquests esdeveniments. L'objectiu principal és facilitar la gestió tant per als administradors com per als clients. Els administradors poden gestionar usuaris, crear, editar i eliminar esdeveniments, i supervisar les vendes d'aquests esdeveniments. D'altra banda, els clients tenen la capacitat de registrar-se, comprar i cancel·lar entrades, així com consultar les seves compres anteriors.

Funcionalitats

  • Header: El capçalera varia segons si l'usuari està loguejat o no, amb opcions de login, registre i visualització de la pàgina d'esdeveniments.
  • Login: Permet als usuaris iniciar sessió amb validacions de nom d'usuari i contrasenya. En cas d'error, es mostra un missatge d'error.
  • Registre: Permet als usuaris registrar-se introduint les seves dades personals, amb validacions i control per evitar registres duplicats.
  • Pàgina d'Esdeveniments: Mostra una vista amb una descripció breu dels esdeveniments disponibles, amb opcions de compra per als clients i informació sobre com registrar-se si no ho han fet encara.
  • Perfil d'Usuari: Proporciona als usuaris una interfície per veure i modificar les seves dades personals, així com cancel·lar entrades.
  • Àrea d'Administració: Ofereix una interfície especialitzada per als administradors, amb funcionalitats de gestió d'usuaris, esdeveniments i entrades associades als esdeveniments.
